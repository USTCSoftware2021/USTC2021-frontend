The model of this attribute is the separable Convolution in the convolutional neural network, and
converts a protein sequence into a one hot form. So want is one hot form? One hot form is a common
way to change a sequence to a pixel-like matrix.

Each columns represent a character, and each character in the sequence corresponds to a row.

In the matrix, the right
position will be placed in 1, and others are 0.

Try the example sequence to experience this process.

D 0 0 0 0 0 0
C 0 0 0 0 0 0
A 0 0 0 0 0 0
B 0 0 0 0 0 0
F 0 0 0 0 0 0
E 0 0 0 0 0 0
