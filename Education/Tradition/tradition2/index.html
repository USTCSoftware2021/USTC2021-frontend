Subcellular Localization

Generally, we use the fluorescent protein in situ identification method to fuse the target gene with the fluorescent protein gene, so that the fusion gene can be expressed in the recepient cells. By observing the display position of the fluorescent protein in the cells, we can locate the target protein, so as to determine the subcellular localization of the target protein. Take the target gene in tobacco cells as an example.
